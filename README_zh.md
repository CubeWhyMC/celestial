# Celestial

[英文/English](./README.md)

> 一个第三方的lunarclient启动器

## 特性

1. 全部开源 (GPLv3)
2. 不使用electron
3. 轻量， 高性能
4. 开箱即用
5. 浏览器登录微软账号支持
6. 跨平台 (*实验*)
7. 自定义API地址 (仅[LunarCN API](https://github.com/CubeWhyMC/website))
8. 多版本,一个Java
9. 支持javaagent (*实验*)
10. 支持LunarCN (*实验*)
11. 支持Weave (*实验*)
